Data Science Symposium - RAG Chatbot

Building a Retrieval-Augmented Generation (RAG) chatbot for the Data Science Symposium. This project implements advanced natural language processing techniques to create an intelligent conversational AI system that can provide accurate, context-aware responses by retrieving relevant information from a knowledge base and generating human-like answers.
